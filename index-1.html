About

Delving deep into different aspects of coding and computers, I have especially taken a liking towards web technologies. The very fact of how Internet works is really fascinating and to be able to create beautiful web pages yourself for the world to see is just an added bonus. I specifically like designing web pages, not only as in how they function but I am very selective about how it should look for me to call it as a good design. This has led me into the never-ending field of front-end designing where new frameworks are designed every day to design your own website. Being a computer science graduate, I am very familiar with the concept of HTML, CSS, Bootstrap and other front end packages like Angular JS and Node JS.
